# 关键词工作台

一个静态本地网页，用来处理 Amazon 关键词的分类、中文解释、关键词处置建议和导出。

## 当前版本范围

- 支持粘贴关键词，一行一个。
- 支持上传 CSV / TSV / TXT。
- 自动识别常见列名：`keyword`、`search term`、`customer search term`、`关键词`、`搜索词`。
- 支持搜索量字段，自动识别 `search volume`、`monthly searches`、`搜索量`、`月搜索量` 等列名。
- 自动给出关键词分类、中文解释、处理动作和状态。
- 支持新增自定义分类，新增后会同步出现在筛选器和每一行的分类下拉中。
- 支持人工修改分类、动作、状态、中文解释和备注。
- 支持保存到本机浏览器。
- 支持导出全部关键词、处理表、否词候选、Listing 埋词建议。
- 关键词表格默认分页渲染，降低大量词表下的交互卡顿。

## 不包含

- 不做广告复盘看板。
- 不接入 Amazon API。
- 不做账号登录、云同步或多人协作。

## 使用方式

直接打开 `index.html` 即可使用。

如果上传到 GitHub Pages，也可以作为纯静态网站运行，不需要后端服务。

## 推荐工作流

1. 上传或粘贴关键词。
2. 检查自动分类结果。
3. 手动修正分类、动作和状态。
4. 导出处理表、否词表或 Listing 埋词表。

## 文件说明

- `index.html`：页面结构。
- `styles.css`：界面样式。
- `app.js`：关键词解析、分类、筛选、保存和导出逻辑。
- `sample-keywords.csv`：带搜索量的示例关键词文件。
